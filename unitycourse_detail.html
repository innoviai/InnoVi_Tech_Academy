About this course
1. Evaluate and identify the characteristics of game engines for game software development;
2. Apply the technical skills in utilizing game engines to develop 2D/3D games;
3. Analyse the interactive games and use the Unity3d Game Engine for game software development;and
4. Analyse the multi-platform games by using the Unity3d Game Engine.
What you'll learn
Module 1 : Game Development Introduction
Overview of game development and the game industry
Introduction of Unity3d game engines and development tools
Introduction to 2D/3D Game Design, Storyboard
Principles of game design and gameplay mechanics
Hands-on exercises to create simple games
Module 2 : Game Basic Features
Character design, environment design, and other visual elements of game development
Game Prototype, game storyboarding
Basic programming concepts and techniques for game development
Physics, collision detection, and other game development concepts
User Interface Design, Navigation and Artifical Intelligence
Module 3 : FPS Game Development
Game Design and Implementation
Principles of level design, Creating AI for enemy characters
Mini-Map, Scoreboard, Networking and Multiplayer Development Tools
Optimization and Performance for FPS Games
Game Controller, Cross-Platforms Development
Learning Outcomes
1. Evaluate and identify the characteristics of game engines for game software development;
2. Apply the technical skills in utilizing game engines to develop 2D/3D games;
3. Analyse the interactive games and use the Unity3d Game Engine for game software development; and
4. Analyse the multi-platform games by using the Unity3d Game Engine.
